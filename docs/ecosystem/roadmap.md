# Roadmap

Short-term roadmap:

1. Containerized server releases through GHCR.
2. Public GitHub Pages docs and landing site.
3. Signed plugin index as marketplace foundation.
4. Signed release manifests for server/agent artifacts.
5. Concrete plugin runner implementation after isolation tests.
6. Geo-Routing apply and parent-zone publication workflow.
7. Log ingestion v2 with per-line accepted offsets.

Long-term direction:

- official sing-box/xray/Sub-Store plugins;
- private DNS deployment and GeoDNS operations;
- richer dashboard UX;
- bbolt-first runtime storage;
- optional proto/gRPC contracts where cross-language clients justify the cost.
